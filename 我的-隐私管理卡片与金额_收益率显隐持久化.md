# 我的-隐私管理卡片与金额/收益率显隐持久化

## Core Features

- 全局默认修改后对所有页面生效（下次启动）

- 覆盖为会话态；默认修改需重启生效

- 页面内小眼睛实时切换（仅当次会话）

- 移除清除覆盖 UI，避免与规则冲突

## Design
