# 投资页三列齐平与报表页布局优化

## Core Features

- 投资页概览三列顶齐（Grid + 列内 flex-start + 不换行）

- 报表顶栏：时间选择器与全局小眼睛同一行，控件宽度收缩不截断

- 分类/账户/标签 TAB：收支切换条与小眼睛同一行，右侧对齐并加 gap

- 修复报表部分 TAB 首次点击小眼睛需两次的问题（消除 setData 竞态）
