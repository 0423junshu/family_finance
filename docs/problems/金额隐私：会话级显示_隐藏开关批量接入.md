# 金额隐私：会话级显示/隐藏开关批量接入

## Core Features

- 为 transfer / transaction-detail / stats 接入小眼睛，会话期有效（不持久化）

- 金额渲染按状态显示或***，不影响原有颜色与布局

- 跨页面返回仍保留本会话选择（依赖 globalData.pageVisibility）
