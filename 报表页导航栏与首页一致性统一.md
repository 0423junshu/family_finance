# 报表页导航栏与首页一致性统一

## Core Features

- 报表页与首页共用 navigation-bar 组件与相同 class/props

- 复用首页 safe-area 与动态顶部 padding 方案（useJsSafeTop）

- 移除报表页对导航栏的本地覆盖，改用统一样式变量

- 保证不同屏幕下响应式行为一致

## Design
