# 金额隐私显示开关与遮罩功能

## Core Features

- 系统设置新增“默认显示金额”全局开关，持久化本地

- 所有金额展示旁增加“小眼睛”按钮，可切换明文/隐藏

- 切换仅影响展示，不影响计算/统计逻辑

- 状态持久化（刷新/重启后保持）

- 前端混淆降低控制台直观可见性

- 提供等价 React 组件与样式，兼容移动端与 PC
